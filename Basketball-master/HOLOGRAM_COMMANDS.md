# Quick Hologram Creation Commands

Copy and paste these commands directly into your server console or in-game (if you have permission).
Replace `world`, `X`, `Y`, `Z` with your actual coordinates.

## Season 1 Leaderboards

### Season 1 Points
```
/dh create season1_points world X Y Z
/dh addline season1_points &6&lSeason 1 Points Leaderboard
/dh addline season1_points &eNext update: %basketball_update_countdown%
/dh addline season1_points  
/dh addline season1_points %basketball_season1_points_1%
```

## Useful Commands

- Move hologram: `/dh movehere <name>` (stand where you want it)
- Edit line: `/dh setline <name> <line#> <text>`
- Remove line: `/dh removeline <name> <line#>`
- Reload all: `/dh reload`
- List all: `/dh list`
- Delete hologram: `/dh delete <name>`
